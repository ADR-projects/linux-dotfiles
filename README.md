## Installation

```bash
git clone https://github.com/ADR-projects/linux-dotfiles.git
cd linux-dotfiles
```

## Bindings

Custom Keybinding for the widget:

```bash
bindd = SUPER, E, Widget, exec, ~/.config/eww/./launch.sh -a
```
